# VenueSync 🎬🏟️

VenueSync is a sleek, modern, dynamic Single-Page Application (SPA) designed to improve the physical event experience for attendees at large-scale venues (movie theaters, sports stadiums, etc.). It addresses crowd movement, wait times, and real-time coordination through a beautifully styled visual dashboard.

## Features ✨
- **Live Multi-Hall Synchronization:** Dynamically tracks multiple events simultaneously (e.g., movies playing in different halls). It calculates exact elapsed times, remaining durations, and flashes real-time statuses like "Intermission."
- **Wait Time Simulator:** Tracks live queues for major facilities (Food Courts and Washrooms) across different venue levels. Wait times automatically fluctuate and change color indicators based on traffic density.
- **Push Notification Analytics:** An intelligent alert engine dynamically generates "Live Updates" for the user whenever their nearest food stand experiences either a rapid line drop or a severe crowd surge.
- **Interactive Multi-View Layout:** Seamlessly switch between the Dashboard, Interactive Map, Food & Drink Pre-order systems, and Venue Info using the sidebar tabs without page reloads.
- **Premium Glassmorphism Aesthetics:** Designed entirely without external frameworks utilizing CSS Grid and modern dark UI patterns. It prominently features neon accents and blurred background layers.

## Tech Stack 💻
- **HTML5:** Core structure and semantic markup.
- **Vanilla CSS3:** Advanced styling (glassmorphism), grids, smooth micro-animations.
- **Vanilla JavaScript:** Real-time data simulation loop, DOM manipulation, formatting, and tab switching. No Node.js constraints!

## Getting Started 🚀
Since VenueSync is constructed using universal web technologies, there are zero server installations or build pipelines required to test it!

1. Open the project folder `\AntiGravity`.
2. Double-click on `index.html` to instantly run the app locally in your default web browser.
3. Wait **8-10 seconds** on the dashboard to experience the real-time simulation logic inject automatic line updates into your notification panel.

*Note: For the best visual experience, use a modern browser like Google Chrome, Microsoft Edge, or Safari.*
